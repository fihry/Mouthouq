## Overview

Mouthouq is a professional services marketplace platform connecting users with vetted household service providers in Morocco. We streamline the process of finding and booking reliable maintenance, cleaning, and home improvement services.

## Technology Stack

### Backend
- Go
- PostgreSQL
- JWT Authentication
- UUID

### Frontend
- Next.js
- Tailwind CSS
- React Query

### Infrastructure
- Docker
- AWS/DigitalOcean
- GitHub Actions

## Development Setup

1. Environment setup:
```bash
cp .env.example .env
# Configure environment variables
```

2. Install dependencies:
```bash
# Backend
cd src/backend
go mod download

# Frontend
cd src/frontend
npm install
```

3. Start development servers:
```bash
# Backend
cd src/backend
go run main.go

# Frontend
cd src/frontend
npm run dev
```
